EMPLOYEES

READ (TODOS):
curl.exe -X GET http://localhost:8080/employees

READ (UNO)
curl.exe -X GET http://localhost:8080/employees/10001

UPDATE:
curl.exe -X PUT http://localhost:8080/employees/10001 -H "Content-Type: application/json" -d --% "{\"birth_date\":\"1953-09-02\", \"first_name\":\"David\", \"last_name\":\"Rivera\", \"gender\":\"M\", \"hire_date\":\"1986-06-26\"}"

CREATE:
curl.exe -X POST http://localhost:8080/employees -H "Content-Type: application/json" -d --% "{\"birth_date\":\"2003-08-05\", \"first_name\":\"Selene\", \"last_name\":\"Ponce\", \"gender\":\"F\", \"hire_date\":\"1996-06-26\"}"

DELETE:
curl.exe -X DELETE http://localhost:8080/employees/500000
